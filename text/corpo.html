/MyPage/

A really simple, small & fast no-database flat file
single page builder (only 4 files). Ideal for a resume, portfolio, 
a simple information page for some event or association.

Completely free to use.

Download now

Caratteristiche 

Built only with PHP and Bootstrap.
No database required!

Create a simple one-page with all your information from a really simple dashboard, insert all your information and save it. You can access to edit all your data from the dashboard without touch the code. Fully responsive, so you can edit the web page directly from your smartphone.

Flat-file

No database to store your data

Bootstrap

Native Bootstrap integration

Dashboard

Edit page data from dashboard

Lightweight

Less than 3mb of space required


How to use it

Just unzip and copy all the zip file content inside the root of your server. No configuration, no code required to start using it.
